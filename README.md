# About this Project

I wanted to translate my google doc resume into markdown because I wanted version control and to see my growth. I want complete creative control, so if I need to do something real specific I can. I found that doc can start breaking when it comes to formating and I found that tedious. 

With the actual files I can actually keep all the files clean on my pc without storing big pdf files. 

All these reasons are nice upsides, but in reality I just thought it would be fun.
